-- API: /pokemon/search (POST)
API Purpose:
Search for Pokémon by ID or name, with optional filters (height, weight, types, stats) and sorting options.

Auth:
User token required.

Inputs:
- searchterm (string): Search keyword for Pokémon name or ID
- filter (object, optional):
  - height (object, optional): min (integer), max (integer)
  - weight (object, optional): min (integer), max (integer)
  - mustHaveTypes (array of integer): Type IDs Pokémon must have
  - mustNotHaveTypes (array of integer): Type IDs Pokémon must NOT have
  - canLearnMoves (array of object): moveId (integer), moveMethodId (integer)
  - stats (array of object): statId (integer), min (integer), max (integer)
- page (integer, default 0): Page number for paginated results
- sortby (integer, default 0): Sorting option (see doc for mapping)

Returns:
- results (array): List of Pokémon matching the search criteria
- totalCount (integer): Total number of Pokémon found
- totalPage (integer): Total number of pages

-- API: /pokemon/move/search (POST)
API Purpose:
Search for Pokémon moves by ID or name, with optional filters (type, power, pp, damage class, priority) and ordering.

Auth:
User token required.

Inputs:
- searchterm (string): Search keyword for move name or ID
- page (integer, default 0): Page number for paginated results
- filters (object, optional):
  - type (integer): Move type ID
  - power (object): min (integer), max (integer)
  - pp (object): min (integer), max (integer)
  - damageClass (integer): Damage class ID
  - priority (object): min (integer), max (integer)
- order (integer, default 0): Ordering option (see doc for mapping)

Returns:
- results (array): List of moves matching the search criteria
- totalCount (integer): Total number of moves found
- totalPage (integer): Total number of pages

-- API: /pokemon/watchlist (POST)
API Purpose:
Add a Pokémon to the user's watchlist by Pokémon ID.

Auth:
User token required.

Inputs:
- pokemonId (number): The ID of the Pokémon to add

Returns:
- id (integer): Unique watchlist entry ID
- user_id (integer): ID of the user
- pokemon_id (integer): ID of the Pokémon
- created_at (string): Timestamp
- deleted (boolean): Soft delete flag

- level (number): Level (1-100)
