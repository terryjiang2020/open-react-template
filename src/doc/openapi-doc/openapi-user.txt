-- API: /user/register (POST)
API Purpose:
Register new user.

Auth:
No token required.

Inputs:
- email (string): User email
- username (string): Username
- password (string): Password

Returns:
- status (number): HTTP status code
- message (string): Registration status
- userId (number): New user ID

-- API: /user/register/verify (POST)
API Purpose:
Verify registration email.

Auth:
No token required.

Inputs:
- email (string): User email
- verificationCode (string): Verification code

Returns:
- status (number): HTTP status code
- message (string): Verification status
- token (string): Auth token
- user (object): User info

-- API: /user/register/verify/resend (POST)
API Purpose:
Resend verification email.

Auth:
No token required.

Inputs:
- email (string): User email

Returns:
- 200 OK if resent
